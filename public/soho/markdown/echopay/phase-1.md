Proves the core tech. In this phase we create a Cloud based backend using Python API with Postgres Database and tsvector to allow uploading and processing of large csv files and superfast searching of that product data

#### Tasks

- Python Environment
- Fast API
- Postgres Database
- CSV upload and process
- Tenant Admin screen to CRUD products in the DB

#### Definition of Done

A working app which tenant can log into and upload CSV files, see them get processed and then manage the Creation, Reading, Updating and Deletion (CRUD) of those products
