Magento 2 payment module integrating **EchoPay** Open Banking payments into the checkout flow.

This module adds EchoPay as a payment method in Magento 2, enabling customers to complete bank-to-bank payments via the EchoPay platform. It handles payment initiation, redirection, callbacks/webhooks, and order state updates.

## Features

- Magento 2.4.x compatible
- EchoPay payment method at checkout
- Sandbox / live environment support
- Secure API authentication (Bearer token)
- Webhook and callback handling
- Multi-store support
- EchoPay transaction IDs stored in admin
- Structured API logging

- Bonus: Addition to Adobe Commerce Marketplace

## Requirements

- Magento 2.4.x
- PHP 8.1+
- EchoPay API credentials
