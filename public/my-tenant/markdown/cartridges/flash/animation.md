> The GreenSock Animation Platform handles the orchestration layer for motion inside Flash

GSAP is a high-performance JavaScript animation library for animating DOM, SVG, canvas and React elements with precise timeline control.

- Drive timeline-based UI animations (enter/exit, transitions, hero motion)
- Coordinate sequenced animations across components
- Keep animation logic declarative but centrally controlled
- Ensure smooth, performant motion without relying on CSS keyframes alone
