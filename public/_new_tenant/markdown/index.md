## What is a tenant?

Imagine you have a shop in a posh shopping centre arcade. You are a tenant. And so are all the other shop owners. 

> How to create new NX tenant

To create a new tenant Duplicate and configure this `_new_tenant `folder, then add it to the library here `/app/NX/lib/getTenant.tsx`
