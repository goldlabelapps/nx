# Shortcodes

Shortcodes are a powerful feature inspired by WordPress, allowing you to embed dynamic content or functionality directly within your text or markdown files. A shortcode is a simple, bracketed tag—such as `[ShortCodeName prop1="hello"]`—that acts as a placeholder for more complex logic, components, or data.

When your content is rendered, the system automatically detects these shortcodes and replaces them with the corresponding output, such as a custom UI element, widget, or formatted data. This keeps your content clean and readable, while enabling advanced features without writing code or HTML directly.

Shortcodes are useful for:
- Embedding interactive components
- Displaying dynamic data
- Reusing content blocks
- Simplifying complex markup

By using shortcodes, you can extend your content with reusable, customizable, and maintainable features, all with a simple and intuitive syntax.
